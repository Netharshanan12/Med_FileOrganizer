About the Tool

The Medical File Organizer CLI Tool is a command-line application designed to organize medical files efficiently. It categorizes files by patient name and file type, ensuring your data is always easy to find and manage.

File Naming Structure

To maximize the tool's effectiveness, ensure your files follow this naming structure:

  • Format: PatientName_FileType_Date.extension
  • Example: JohnDoe_Report_2025-01-26.pdf

Files that do not follow this structure will still be organized but may be categorized under "Unknown_Patient."

How to Use

  1. Download the tool as a binary for your platform (Linux, Windows, or MacOS).
  2. Place the executable in your desired directory.
  3. Run the tool using the following syntax: 
    ./Med_FileOrganizer --directories DIR1,DIR2
  4. Replace DIR1 and DIR2 with the paths to your directories.

Features

  • Supports multiple directories.
  • Handles duplicate files with unique naming.
  • Categorizes files into types such as Imaging Files, Patient Reports, Lab Reports, and Others.
  • Generates organized folders automatically.
